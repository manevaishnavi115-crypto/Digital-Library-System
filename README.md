# Digital Library Management System

## Project Description

This project is a **Digital Library Management System** developed using HTML, CSS, and JavaScript.
It allows users to browse books, magazines, and reference materials in a digital format.

## Features

* View digital books
* Browse magazines
* Access reference materials
* Simple and user-friendly interface
* Organized library resources

## Technologies Used

* HTML
* CSS
* JavaScript
